# Sistema de reservas

Aplicación web estática para gestionar reservas de servicios por fecha y hora.

## Tecnologías
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Selección de fecha
- Horarios disponibles y no disponibles
- Prevención de doble reserva
- Persistencia en localStorage
- Modal para agenda
- Listado de reservas del día
- Panel admin básico

## Reglas del negocio
- Apertura: 8:00 a. m.
- Cierre: 10:00 p. m.
- Duración del servicio: 1 hora
- Último servicio agendable: 9:00 p. m.

## Ejecución
1. Abrir la carpeta en Visual Studio Code.
2. Abrir `index.html` en navegador.
3. Recomendado usar Live Server.
